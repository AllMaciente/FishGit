Atalhos Git para Fish Shell.

|-------------|---------------------------------|----------------|
| `gi <linguagem>` | Cria um `.gitignore` personalizado | Usa a API do Toptal para baixar um `.gitignore` específico para a linguagem ou tecnologia informada. |
